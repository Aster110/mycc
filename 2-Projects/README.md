# 2-Projects 项目

存放正在进行的项目。

## 目录结构建议

```
2-Projects/
├── P01-项目名/
│   ├── 00-项目概览.md
│   ├── 01-需求.md
│   ├── 02-执行记录.md
│   └── ...
├── P02-另一个项目/
└── completed/          # 已完成的项目
```

## 命名规则

- `P01`、`P02`... 编号便于排序和引用
- 项目名简短明确

## 项目概览模板

```markdown
# P01 - 项目名

> 创建时间：YYYY-MM-DD
> 状态：规划中 / 进行中 / 已完成

## 一句话描述

这个项目是做什么的。

## 目标

- [ ] 目标 1
- [ ] 目标 2

## 下一步

当前最重要的下一步行动。
```
