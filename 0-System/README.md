# 0-System 系统核心

这个目录是 CC 系统的"大脑"，存放状态和记忆相关的文件。

## 文件说明

| 文件 | 用途 | 更新频率 |
|------|------|----------|
| `status.md` | 短期记忆，当前状态快照 | 每次对话可更新 |
| `context.md` | 中期记忆，本周每日快照 | 每日睡前追加 |
| `templates/` | 日/周模板 | 基本不变 |

## 记忆系统工作原理

```
每次对话开始
    │
    ▼
Hooks 自动读取 status.md
    │
    ▼
注入到 CC 上下文（短期记忆）
    │
    ▼
CC 了解你的当前状态
```

## 使用方法

1. **首次使用**：复制 `status.md` 并填写你的初始状态
2. **日常使用**：CC 会根据对话自动更新 `status.md`
3. **睡前**：把当天 status 关键信息追加到 `context.md`
4. **周末**：复盘本周，将 `context.md` 归档到 `5-Archive/周记/`

## 自定义扩展

你可以添加更多状态文件，比如：
- `about-me/`：存放你的个人画像、能力结构、价值观等长期记忆
- `goals.md`：长期目标追踪
- `habits.md`：习惯追踪

记得在 `.claude/settings.local.json` 的 hooks 中添加对应的读取命令。
