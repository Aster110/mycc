# About Me 个人画像

存放关于你自己的信息，帮助 CC 更好地理解你。

## 建议包含的内容

- **基本信息**：名字、职业、所在城市
- **工作风格**：偏好的沟通方式、决策风格
- **能力结构**：擅长什么、不擅长什么
- **价值观**：什么对你重要
- **目标**：短期/长期目标
- **习惯**：作息、偏好

## 文件结构建议

```
about-me/
├── profile.md        # 基本画像
├── strengths.md      # 优势和擅长
├── goals.md          # 目标
└── preferences.md    # 偏好设置
```

## 为什么需要这个

CC 读取这些信息后，能更准确地：
- 给出符合你风格的建议
- 理解你的上下文
- 避免重复询问基本信息

## 隐私提醒

这个目录包含个人信息，默认被 `.gitignore` 忽略，不会上传到公开仓库。
